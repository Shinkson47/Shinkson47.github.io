Timeline

Milestones of Library Version 2.
A visualisation of the painfully slow progress.

March 2020
2020 Temporary Hiatus

Active production halted. Expected to return around september as the events of 2020 settle, and campus' reopen.

9 May 2020
OPEX Visual Conception

Created a basic GUI tool for the JGEL toolbox, providing a simple way to generate and regenerate console instruction classes. Uses Javapoet to generate classes to place directly in your project, and serializes command JGELCMD files to make changes to exsisting commands and recompile.

8 May 2020
Project refactorisation

Project JGEL was finally given the dedicatated name "Phoenix" leading to the refactorisation of the project to "Open Phoenix Engine", or shortened as OPEX.

10 April 2020
Instruction Builder

Created a basic GUI tool for the JGEL toolbox, providing a simple way to generate and regenerate console instruction classes. Uses Javapoet to generate classes to place directly in your project, and serializes command JGELCMD files to make changes to exsisting commands and recompile.

3 April 2020
Console Interpreter

Introduced the framework basis for the engine's console interface, accessable via the runtime terminal. It uses classes for instructions, and is entirely expandable. Also theorised an instruction builder tool.

31 March 2020
JGEL Webpage born

During a much needed revamp to shinkson47.in, the original version of the very page you're looking at right now was added. It's since been overhauled once more since shinkson47.in was re-developed with Bootstrap 4.

17 January 2020
Version 1 Dropped

JGEL V1 Was pushed for the final time. For many reasons, but mostly because the project has little pre-thought, and was becomming more unorganised and unprofessional by the commit. I have plans for how i want JGEL to function, and be interacted with, and this implementation is not addequate. However it paved the way for multiple systems, and provide a good insight to what this project would be like.
