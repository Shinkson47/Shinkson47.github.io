WEAPONS OF CHOICE

Oracle's Java

One of the worlds largest names in robust, cross platform, object oriented, functional programming, widely used.

Built to Java Bytecode, it's the default language for the JRE VM.
Java

Microsoft's Visual C#

Two years younger than Java, c# is a mature yet up-to-date multi-paradigm, general purpose language developed and maintained by Microsoft.

Careful, it's sharp!

Google's Dart

Developed by Google, Dart is a 'Write once, run everywhere' object oriented scripting language, capable of running natively in Dart VM on ARM32, ARM64, x86_64, and on Javascript runtimes.
Dart

JetBrains' Kotlin

Built on the cross platform JRE, kotlin is a Java Bytecode compatable language built to match the functionality of Java in a significantly more simple and streamline way.
Kotlin
