WEAPONS OF CHOICE

Oracle's Java

Used as the base of all of my largest projects to date, Java is one of the worlds largest names in robust, strong typed, cross platform, object oriented, functional programming.  Built to Java Bytecode, it's the default language for the JRE VM. 
Java

Microsoft's Visual C#

C# is the langauge of chouce for all projects i've worked on for the Unity Engine. Two years younger than Java, C# is a mature yet up-to-date multi-paradigm, general purpose language developed and maintained by Microsoft.

Careful, it's sharp!

Google's Dart

A newer name that's extremely useful for multipurpose scripting, or developing Flutter applications. It's a  scripting language with object oriented capabilities developed by Google following the 'Write once, run everywhere' philosophy, capable of execution on ARM32, ARM64, x86_64, and Javascript VM.
Dart

JetBrains' Kotlin

Used to streamline Java objects in OPEX, kotlin is a Java Bytecode compatable language built to match the functionality of Java in a significantly cleaner way, with less boilerplate and overhead.
Kotlin
