Small scale projects

Splash

Now discontinued, Splash XX was a 2 year long personal development project to develop a game, seeing 5 majour re-writes, each significantly different. 

The project started from a conceptual rendering framework built in .NET, expanding to see what it could be used to create. Many itterations of the project saw many different game concepts and back-end system being developed in VB.NET, before switching to Java for the final two versions.

Pixel fighting

Rendering proof of concept build on the .NET and Metro frameworks.

Built to satisfy an idea of a colonial territory fighting simulator. A fairly complete project, asside from a few small front end flaws.
